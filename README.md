# Sistema Acadêmico - Algoritmos 2

Projeto desenvolvido para a disciplina de **Algoritmos 2** envolvendo estruturas de dados e algoritmos em C.

## Funcionalidades
- ✅ Cadastro de alunos e disciplinas
- ✅ Gestão de matrículas  
- ✅ Controle de compromissos
- ✅ Alocação dinâmica de memória
- ✅ Relatórios com ordenação (qsort)

## Estrutura
- `cadastroDisciplinas.h` - Definições de structs e protótipos
- `structsDisciplinas.c` - Implementação das funções
- `main.c` - Programa principal
